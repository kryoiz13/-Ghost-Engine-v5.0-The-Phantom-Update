## üïπÔ∏è How It Works

1.  **Input:** Paste any YouTube URL (Music Video) into the dashboard.
2.  **Extraction:** The engine scrapes the Closed Captions (CC) directly from YouTube.
3.  **Refinement:** The bot cleans the text, removing timestamps and non-lyric tags.
4.  **The Hand-off:** You get a **5-second window** to focus your WhatsApp chat window before typing begins.
